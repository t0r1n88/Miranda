Описание задачи
Дано:
Большое количество файлов excel представляющих собой таблицы  с заголовком в виде объединеной ячейки.

Требуется:
Объединить эти файлы в один файл, где каждый файл будет расположен на отдельном листе.Каждый лист должен быть назван по названию заявки.


Инструменты:
pandas
библиотеки для работы с excel

Предполагаемый алгоритм:
1) Считать первую строку файла
2) Извлечь оттуда номер заявки
3) Создать пустой документ
4) Создать лист с номером заявки
5) Записать из считанного файла всю таблицу в лист с соответсвующим номером заявки
6) Сохранить файл

Возможные проблемы
1) Первая строка  каждой таблицы представляет собой объедененную ячейку, пандас может плохо работать с таким, а значит нужно будет использовать другую библиотеку
2) в некоторых файлах название таблицы находится не на первой строке
Возможные решения
Перебор строк файла пока не найдется первая заполненная строка
3) Названия таблиц в некоторых файлах имеют вид  Запрос по профобучению б/граждан от ЦОПП на 09.09. 2020 г. А не стандартный в виде Заявка №45.

Подготовительный этап
1) Получить список всех файлов экселя с помощью функции разработанной для проекта dodger
2) Определить может ли пандас работать с обьединой ячейкой.
